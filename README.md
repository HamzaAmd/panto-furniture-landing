# Panto Furniture Landing

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/HamzaAmd/panto-furniture-landing.git
   cd panto-furniture-landing
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Approach

The project is built using a combination of modern web technologies including HTML, CSS, and JavaScript. The primary focus is on creating a responsive and user-friendly landing page for a furniture store. Key features include:
- Intuitive navigation
- Interactive product displays
- User-friendly interface

## Architecture

The architecture of the application follows a component-based structure, which allows for reusability and maintainability. Key components include:
- Header
- Product List
- Footer

Additionally, we employ a simple state management system to handle user interactions and data flow.
